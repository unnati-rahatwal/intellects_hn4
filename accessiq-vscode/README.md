# AccessIQ Remediation Sync

This VS Code extension allows you to import JSON audit reports generated by the AccessIQ platform and automatically apply AI-driven accessibility remediations directly to your workspace files.

## Features
- **Import JSON Reports**: Easily ingest `a11y-audit.json` files containing detailed violation data.
- **AI Remediation Sync**: Review fixable violations and automatically apply the recommended AI patches surgically to your HTML/JSX templates.

## Usage
1. Open the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`).
2. Type `AccessIQ: Import Remediation Report`.
3. Select your downloaded JSON report.
4. Choose a violation from the dropdown menu to apply its fix to your codebase.
